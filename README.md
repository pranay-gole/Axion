# 🚀 Cloud Gaming Platform (Flask Based)

A futuristic cloud gaming web platform built using Flask, HTML5 Canvas and JavaScript.

## Features

- 🔐 Login & Registration system
- 🎮 Interactive Dashboard
- ⚡ Axion Runner Game
    - Neon animated character
    - Gradually increasing difficulty
    - Small + Big spike obstacles
    - Persistent Best Score
    - Light / Dark Theme toggle
- 📱 Responsive full-screen gameplay
- 🌌 Cyberpunk-inspired UI

## Tech Stack

- Python (Flask)
- HTML5 Canvas
- JavaScript
- CSS
- LocalStorage

## Controls (Axion Runner)

- SPACE → Jump
- R → Restart
- Theme button → Toggle Day/Night
